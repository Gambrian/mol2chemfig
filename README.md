# Chemfig 实验室 v0.0.3

这是一个面向非程序用户的轻量级桌面/网页工具，用于将化学 SMILES 字符串或 Molfile (V2000) 转换为 LaTeX `chemfig` 代码，并提供本地实时预览。

## 功能特点

- **双入口转换**
  - **SMILES 转换**: 输入 SMILES 即可生成 `chemfig`。
  - **Molfile 转换**: 支持粘贴 V2000 Molfile 文本进行转换。
- **双结果展示**
  - **预览优化结果**: 为当前页面的轻量预览做了兼容清理，可直接修改并实时渲染。
  - **原始转换结果**: 保留转换器返回的原始 `chemfig`，适合复制到 LaTeX 文档进一步使用。
- **双重参考**
  - **RDKit 结构参考**: 作为分子结构基准图。
  - **本地 Node 渲染预览**: 使用 `node-tikzjax` 生成 SVG，不依赖完整 LaTeX 安装。
- **面向分发的轻量设计**
  - 页面操作尽量简单，适合没有代码基础的使用者。
  - 不引入更重的完整 LaTeX 渲染链，降低安装和运行负担。

## 安装要求

### 1. Python 环境
建议使用 Python 3.10+。安装以下依赖：

```bash
pip install -r requirements.txt
```

### 2. Node.js 环境 (本地渲染引擎)
项目使用轻量的本地 Node 渲染引擎来生成 SVG 预览，推荐 Node.js 16+。

```bash
npm install
```

如果使用 `gui_launcher.py`，程序会优先检测系统 Node.js；未安装时可按提示下载便携版 Node。

## 使用说明

### 1. 启动服务
```bash
python app.py
```
默认运行地址：`http://127.0.0.1:5000`

也可以使用桌面管理器：

```bash
python gui_launcher.py
```

### 2. 操作流程
1. 访问首页进入 **Molfile 转换** 页面，或点击右上角切换到 **SMILES 转换**。
2. 输入内容并点击转换。
3. 页面会同时给出：
   - **预览优化结果**：用于当前页面实时渲染。
   - **原始转换结果**：保留原始 `chemfig` 输出，适合复制到 LaTeX 文档。
4. 右侧 SVG 预览支持滚轮缩放、拖拽平移、手动刷新和视角重置。

### 3. LaTeX 配置
在 LaTeX 文档中使用生成代码前，请确保包含以下宏包：
```latex
\usepackage{chemfig}
```

## 技术说明

- **后端 (Flask)**
  - 负责 SMILES/Molfile 转换。
  - 使用 `mol2chemfigPy3` 生成 `chemfig`。
  - 使用 `RDKit` 生成结构参考图。
- **本地预览引擎 (Node.js)**
  - 通过 `render_tikz.js` 调用 `node-tikzjax`。
  - 预览链路保持轻量，但兼容性低于完整 LaTeX 环境。
- **前端 (HTML + JS)**
  - 支持实时预览、防抖渲染、复制代码、缩放和平移。
  - 同时展示原始输出和预览优化输出，降低误用风险。

## 注意事项

- **原始结果与预览结果不同是正常现象**
  - 原始结果更适合导出到 LaTeX。
  - 预览结果会针对当前轻量渲染器做兼容清理。
- **Node 依赖**
  - 如果没有执行 `npm install`，右侧 SVG 预览将不可用，但转换功能仍可工作。
- **兼容性边界**
  - 当前预览引擎不是完整 LaTeX 环境，部分复杂 `chemfig` 宏仍可能无法预览。
- **渲染耗时**
  - 当前版本不主动设置本地预览超时，低性能机器上复杂结构可能需要较长等待时间。
- **错误日志**
  - 预览失败时会显示更完整的后端日志，包含预览代码和展开后的 TeX 内容，便于继续定位问题。
