# Changelog

## 0.1.3 - 2026-03-12

- 新增“原始转换结果”展示，与“预览优化结果”分开呈现，便于导出和比对。
- 优化 SMILES/Molfile 页面布局，减少非技术用户的操作歧义。
- 修复首页路由，默认进入 Molfile 转换页面。
- 改进 Node 渲染调用方式，改为通过标准输入传递 `chemfig`，提升长文本稳定性。
- 取消本地预览超时限制，并补充更完整的渲染日志输出。
- 新增核心错误摘要提取，预览失败时会优先显示首个关键 TeX 报错。
- 修复 `node-tikzjax` 失败时只返回通用错误的问题，补充底层 TeX log 与 TeX 源输出。
- 为轻量预览增加自由基点兼容处理，`\\charge{90=\\.}{C}` 会在预览层自动转换为可渲染形式。
- 优化资源路径与便携 Node 检测逻辑，改善分发环境下的稳定性。
- 修复打包脚本中的 Indigo 动态库硬编码路径问题。

## 0.1.2

- 改善 R 标签在 `chemfig` 中的展示效果。
