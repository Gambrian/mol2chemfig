# Chemfig 实验室 - Windows 图形化便携版

本工具是一个“开箱即用”的化学结构转换与预览工具。现在已集成 **图形化管理器**，支持环境自动检测与一键启动。

## 🚀 快速启动

1. **运行管理工具**：双击运行 `ChemfigLab.exe`。
2. **环境自检**：
   - 启动后，程序会自动检测您的电脑是否有 Node.js 环境。
   - **自动安装**：如果未检测到 Node.js，程序会询问是否从国内镜像站（npmmirror）下载便携版。点击“是”即可自动完成配置。
3. **启动服务**：点击界面上的 **“启动服务”** 按钮。
4. **开始使用**：服务启动后，点击 **“在浏览器打开页面”** 即可开始转换 SMILES/Molfile。

---

## 🛠️ 管理器功能

- **实时日志**：在界面中心的黑色区域，您可以实时看到 Flask 后端的运行状态和错误信息。
- **环境修复**：自动处理 `node_modules` 依赖安装。
- **一键控制**：支持随时停止服务或重启。
- **托盘管理**：直接关闭管理器窗口将同时安全停止所有后台服务。

---

## 📝 核心功能说明

- **SMILES 转换**：输入 SMILES 字符串（如 `c1ccccc1`）即可。
- **Molfile 转换**：支持粘贴 V2000 格式的 Molfile 内容。
- **高清预览**：集成 TikZJax 矢量渲染，支持鼠标缩放和日志查看。

---

## ⚠️ 注意事项

1. **防火墙提示**：初次启动时，Windows 可能会弹出防火墙警告，请选择“允许访问”，否则浏览器无法连接到本地 5000 端口。
2. **端口占用**：如果 5000 端口已被其他程序占用，请关闭冲突程序后重新启动。
3. **复杂语法**：内置的 TikZJax 引擎为了轻量化，仅支持核心 Chemfig 语法。复杂的 LaTeX 宏包可能无法在预览中渲染，但生成的代码仍可在完整 LaTeX 环境中使用。
